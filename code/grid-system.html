Struktur des Grid

<div class="grid-row">
  <div class="lg-cell-1"> "Content" </div>
</div> 
    Die Cells Divs muessen immer von dem Div mit der Klasse grid-row umschlossen werden.
    
    Bei nicht vergabe von kleineren klassen wie .md-cell- oder .sm-cell- wird am Breakpoint immer auf 100% Breite Skaliert.

Offset von Cells

<div class="grid-row">
  <div class="lg-cell-1 lg-cell-offset-2"> "Content" </div>
</div> 
    Um eine Cell einzuruecken muss einfach die jeweilige klasse hinzugefuegt werden. lg-cell-offset-.
